## 样式约束

- 禁止硬编码颜色值（如 `#333`, `rgb(...)`），必须用 CSS 变量
- 禁止 magic spacing（如 `p-[17px]`），用标准 Tailwind scale
- 亮/暗模式：通过 CSS 变量自动切换，不需要手动处理
